💾 QUICK START COMMANDS

  # Start container (interactive bash)
  docker compose up

  # Hot-reload development mode
  docker compose up --watch

  # Run phase tests
  docker compose exec qrft-toolkit python -m tests.test_synthetic
  docker compose exec qrft-toolkit python -m tests.test_phase3
  docker compose exec qrft-toolkit python -m tests.test_phase4

  # Run EHT pipeline
  docker compose exec qrft-toolkit python run_eht_pipeline.py

  # Jupyter Lab (optional)
  docker compose --profile jupyter up
  # Visit: http://localhost:8888 (token: qrft_token_2025)

  # View logs
  docker compose logs -f qrft-toolkit

  # Interactive shell
  docker compose exec qrft-toolkit bash

  # Copy results
  docker cp qrft-blackhole-toolkit:/app/output ./results

  # Stop container
  docker compose down

  # Full cleanup
  docker compose down -v

🎯 GPU DEPLOYMENT

  For NVIDIA GPU systems:

  1. Edit .env:
     ENABLE_GPU=true
     CUDA_VERSION=12.x  (or 11.x)

  2. Edit docker-compose.yml runtime:
     runtime: nvidia

  3. Rebuild and run:
     docker compose up --build

  Expected speedup: ~100x (EVPA kernel, 27.4M → 2.7B px/s)
