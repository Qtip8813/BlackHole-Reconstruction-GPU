2. Start the container
   docker compose up

3. Run the pipeline
   docker compose exec qrft-toolkit python -m tests.test_phase3

4. Process EHT data
   docker compose exec qrft-toolkit python run_eht_pipeline.py

5. View results
   docker compose exec qrft-toolkit ls -la /app/output/
